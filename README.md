# 工作檢核表 2026

## 📋 檔案說明

本專案產生三種不同的檢核表 HTML 檔案：

### 1. checklist_2026_full.html (220KB)
完整的互動式檢核表，涵蓋 2026 年 3 月至 12 月
- **產生方式**：
  ```bash
  python3 generate_checklist.py
  python3 fix_alignment.py
  ```
- **內容**：每日、每週、每月任務

### 2. checklist_2026_periodic.html (16KB)
週期性任務檢核表，單頁年度視圖
- **產生方式**：
  ```bash
  python3 generate_checklist_periodic.py
  ```
- **內容**：每季、每半年、每年任務

### 3. bank_audit_checklist.html (16KB)
銀行稽核檢核表
- **產生方式**：手動調整製作
- **內容**：稽核專用檢核項目

## ✨ 主要功能

### 1. 三欄佈局

#### 左欄：每日任務 (300px)
- 以**月度**方式呈現，不需每日重複勾選
- 包含三項每日必做任務：
  - 理專報表: PRT_D016500
  - 理專報表: WMD0002(錄音)
  - 每日出勤檢視(PeopleSoft)

#### 中欄：每月任務 (350px)
- 依截止日期分類：
  - 【10號前】- 自行查核相關任務
  - 【15號左右】- 金融卡自檢、存單未領回
  - 【月底】- 綜合報告、輪值KEY
  - 其他月報表
- 特殊月份任務標示 ⭐ (黃色高亮)

#### 右欄：每週任務 (剩餘空間)
- 按週次排列（第1週至第5週）
- 每週固定四項任務：
  1. 抽查櫃員現金二次
  2. 抽查外匯現金一次
  3. 抽查保管箱錄影帶一次
  4. 抽查金庫現金500元以上一次

### 2. 列印功能 🖨️

- 每個月份左上角都有**「列印本月」按鈕**
- 點擊後僅列印該月份
- 自動設定為 **A4 橫式** 格式
- 列印時隱藏其他月份和按鈕

### 3. 特殊月份任務

根據 doc.md 規則自動標示：

| 月份 | 特殊任務 |
|------|---------|
| 1月、7月 | 理專高風險投資聲明書 |
| 1、4、7、10月 | 最適員額、內控聲明書 |
| 3、6、9、12月 | 安全維護檢查報告表 |

### 4. 互動功能

- ✅ 可勾選 checkbox
- ✅ 自動儲存勾選狀態 (localStorage)
- ✅ 勾選後自動刪除線
- ✅ Hover 視覺效果
- ✅ 響應式設計

## 🚀 使用方式

1. 在瀏覽器中開啟對應的 HTML 檔案：
   - `checklist_2026_full.html` - 每日/每週/每月任務
   - `checklist_2026_periodic.html` - 每季/每半年/每年任務
   - `bank_audit_checklist.html` - 稽核專用檢核表
2. 勾選完成的任務
3. 需要列印時：
   - 完整檢核表：點擊該月的「列印本月」按鈕（單月列印）
   - 週期性檢核表：點擊「列印」按鈕（整頁列印）
4. 所有勾選狀態會自動儲存在瀏覽器中

## 🔧 重新生成檔案

### 完整檢核表 (checklist_2026_full.html)
```bash
python3 generate_checklist.py
python3 fix_alignment.py
```

### 週期性任務檢核表 (checklist_2026_periodic.html)
```bash
python3 generate_checklist_periodic.py
```

### 稽核檢核表 (bank_audit_checklist.html)
此檔案為手動調整製作，如需修改請直接編輯 HTML 檔案。

## 📅 涵蓋範圍

- **checklist_2026_full.html**：2026年3月至12月，共10個月份
- **checklist_2026_periodic.html**：2026年全年度週期性任務
- **bank_audit_checklist.html**：銀行稽核專用項目

## 💾 資料儲存

- 使用瀏覽器 localStorage 儲存勾選狀態
- 清除瀏覽器資料會重置所有勾選
- 每個 checkbox 都有唯一 ID 確保正確儲存

## 📄 列印設定

- 紙張大小: A4
- 方向: 橫向 (Landscape)
- 邊距: 1cm
- 每次只列印一個月份
- 自動隱藏不需要的元素（按鈕、背景等）
