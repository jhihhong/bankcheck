# 工作檢核表 2026 (3月-12月)

## 📋 檔案說明

- **checklist_2026_full.html** - 完整的互動式檢核表 (90KB)
- **generate_checklist.py** - Python 生成器腳本
- **doc.md** - 原始任務需求文件

## ✨ 主要功能

### 1. 三欄佈局

#### 左欄：每日任務 (300px)
- 以**月度**方式呈現，不需每日重複勾選
- 包含三項每日必做任務：
  - 理專報表: PRT_D016500
  - 理專報表: WMD0002(錄音)
  - 每日出勤檢視(PeopleSoft)

#### 中欄：每月任務 (350px)
- 依截止日期分類：
  - 【10號前】- 自行查核相關任務
  - 【15號左右】- 金融卡自檢、存單未領回
  - 【月底】- 綜合報告、輪值KEY
  - 其他月報表
- 特殊月份任務標示 ⭐ (黃色高亮)

#### 右欄：每週任務 (剩餘空間)
- 按週次排列（第1週至第5週）
- 每週固定四項任務：
  1. 抽查櫃員現金二次
  2. 抽查外匯現金一次
  3. 抽查保管箱錄影帶一次
  4. 抽查金庫現金500元以上一次

### 2. 列印功能 🖨️

- 每個月份左上角都有**「列印本月」按鈕**
- 點擊後僅列印該月份
- 自動設定為 **A4 橫式** 格式
- 列印時隱藏其他月份和按鈕

### 3. 特殊月份任務

根據 doc.md 規則自動標示：

| 月份 | 特殊任務 |
|------|---------|
| 1月、7月 | 理專高風險投資聲明書 |
| 1、4、7、10月 | 最適員額、內控聲明書 |
| 3、6、9、12月 | 安全維護檢查報告表 |

## 🚀 使用方式

1. 開啟 `checklist_2026_full.html` 在瀏覽器中
2. 勾選完成的任務
3. 需要列印時，點擊該月的「列印本月」按鈕
4. 所有勾選狀態會自動儲存在瀏覽器中

## 🔧 重新生成

如需修改內容，編輯 `generate_checklist.py` 後執行：

```bash
python3 generate_checklist.py
```

## 📅 涵蓋範圍

2026年3月至12月，共10個月份的完整檢核表。

## 💾 資料儲存

- 使用瀏覽器 localStorage 儲存勾選狀態
- 清除瀏覽器資料會重置所有勾選
- 每個 checkbox 都有唯一 ID 確保正確儲存

## 📄 列印設定

- 紙張大小: A4
- 方向: 橫向 (Landscape)
- 邊距: 1cm
- 每次只列印一個月份
- 自動隱藏不需要的元素（按鈕、背景等）
